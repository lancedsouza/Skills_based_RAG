AI Skill Intelligence EngineDeveloped by:Lancelot D modern, full-stack recruitment tool designed to solve the "Keyword Gap." Instead of relying on exact text matches, this engine uses Vector Embeddings to understand the semantic meaning of professional skills, providing "Zero-Input" discovery for recruiters.✨ Core FeaturesSemantic Autocomplete: Real-time suggestions that account for typos (e.g., "receuitemnt" → "Recruitment") using high-dimensional vector space.Zero-Input Discovery: Once a skill is selected, the AI automatically triggers a search for "Neighboring" skills without the user needing to type.Vector Search Engine: Powered by pgvector in PostgreSQL to calculate cosine similarity between 1,024-dimensional embeddings.Hybrid Search Logic: Combines traditional keyword matching with deep-learning semantic search.🛠️ Tech StackLayerTechnologyPurposeFrontendNext.js 14 (React)Tailind CSS, Client-side State ManagementBackendFastAPI (Python)High-performance asynchronous APIDatabasePostgreSQL + pgvectorStoring and querying vector embeddingsAI ModelOllama (mxbai-embed-large)Generating 1024-dimension skill embeddingsDevOpsDocker & Docker ComposeContainerized microservices architecture📐 How It WorksThe system translates human language into mathematical coordinates.Ingestion: Professional skills (ESCO dataset) are processed through Ollama to generate vectors.Selection: When a user selects a skill, the frontend sends the array of selected skills to the /multi-skill-search endpoint.Vector Arithmetic: The backend calculates the average vector of the selection:$$\vec{V}_{query} = \frac{1}{n} \sum_{i=1}^{n} \vec{v}_i$$Discovery: The database returns the $K$-nearest neighbors (KNN) based on the shortest angular distance.🚀 Getting Started1.
